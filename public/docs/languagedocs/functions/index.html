Functions

Mutant supports user defined functions. Let’s see them in action in below examples:

Example 1
let sum = fn(n1, n2) {
    return n1 + n2;
};

sum(10, 20);
Example 2
let minus = fn(n1, n2) {
    putln("In minus function");
    putln(n1 - n2);
}

minus(20, 10);
