[Mutant](https://github.com/gaurav-gogia/mutant) supports variable declaration and reassignment. Let's see them in action in below examples:

###### Example 1
```js
let name = "sakamoto";
```

###### Example 2
```js
let age = 20;
```

###### Example 3
```js
let isCat = true;
puts(isCat);

let isCat = false;
puts(isCat);
```

###### Example 4
```js
let array = [1, "sakamoto", true, "cat"];
```

###### Example 5
```js
let map = {
    "name": "sakamoto",
    "age": 20,
    "isCat": true,
};
```
